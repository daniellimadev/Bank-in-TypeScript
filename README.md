# Bank in TypeScript

## Technologies used

* **Language:** TypeScript
* **Versioning:** Git/GitHub


<br>

## Description

* This bank project was made to practice ``OOP`` in TypeScript, the concepts of ``inheritance``.

### How to run the project

1. Clone the repository

2. Install the dependencies
    
     ``npm install``

3. Execute the project

     ``npm run dev``
